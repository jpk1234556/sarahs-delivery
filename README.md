# Sarah's Delivery Website

A professional website for Sarah's Delivery company featuring:

## Pages
- Home (index.html) - Main landing page
- Services (services.html) - Details of delivery services offered
- About Us (about.html) - Company history and team information
- Contact (contact.html) - Contact form and company information
- Testimonials (testimonials.html) - Client feedback and reviews
- Gallery (gallery.html) - Photo gallery of services and team
- Portfolio (portfolio.html) - Professional portfolio for Sarah Ingabire

## Features
- Responsive design that works on mobile, tablet, and desktop
- Modern CSS styling with gradients and animations
- Interactive JavaScript elements including mobile navigation
- SEO optimized with meta tags, sitemap, and robots.txt
- Image gallery with filtering capabilities
- Contact form with validation

## Deployment
Simply upload all files to your web hosting provider. The website uses only HTML, CSS, and JavaScript with no backend requirements.

## File Structure
```
├── index.html (Home page)
├── services.html
├── about.html
├── contact.html
├── testimonials.html
├── gallery.html
├── portfolio.html
├── 404.html (Error page)
├── styles.css (Main stylesheet)
├── script.js (JavaScript functionality)
├── robots.txt (SEO)
├── sitemap.xml (SEO)
├── favicon.ico (Website icon)
├── images/ (Image folder with all photos)
└── README.md (This file)
```

## Customization
To customize the website:
1. Replace images in the `images/` folder with your own photos
2. Update contact information in `contact.html`
3. Modify content in each HTML file as needed
4. Update meta tags in the `<head>` section of each HTML file
